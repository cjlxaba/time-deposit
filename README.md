# Time Deposit Refactoring Kata - Take-Home Assignment

## XA Bank Time Deposit

### Context
STACK:
 Java 17
 SPRING BOOT 3
 MAVEN
 H2 DB ( IN MEMORY DB FOR TESTING PURPOSE) 

HOW TO RUN:
================
navigate to the project directory : time-deposit-take-home-kata/java
then execute :   mvn spring-boot:run 
SWAGGER ENDPOINT : http://localhost:8080/swagger-ui/index.html
H2 database url : http://localhost:8080/h2-console/
DB details : application.properties inside the project under resources
GIT Repo Link: 
 
NB:  To testing purposes , the application auto data.sql (under resources file) to insert test data (time deposits with their withdrawals)
